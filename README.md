# Terra App API

Server-side application supporting The **Terra** App.

See the Client Repo here:  https://github.com/mblakley4/terra-app

Visit the Live Site: https://terra-app.now.sh


# Technology Used:
  - JavaScript
  - Node.JS w/ Express
  - PostgreSQL
  - Deployed on Heroku
